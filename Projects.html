Word frequency analyzer/comparator

For my data structures class, I created a program (with a partner) that compares word frequencies in given text files to compute a value representing the similarity of two texts. This program can be used to predict who wrote a particular text, by comparing its word-frequency "signature" to other texts by the candidate authors. This involved implementing and using a variety of data structures (including a self-balancing binary tree, hashtable, a modified form of a linked list, and a max-heap) and sorting algorithms (such as merge sort, insertion sort, and heap sort).

Search engine and web server

I developed a file search engine in C/C++, as well as a web server to process user queries using the search engine. To speed up query processing, I implemented a hash table in C, and then used an inverted-index to pre-process the files to be searched into index files. This involved a fair amount of low-level pointer manipulation.
