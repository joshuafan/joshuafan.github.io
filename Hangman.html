Hangman Game

In this game, the goal is to try to guess the word the computer is thinking of by guessing letters. If the word has the letter, the location of that letter is revealed. However, if you guess incorrectly too many times, you lose.

I realize that the behavior at the end of the game is suboptimal because it allows the user to continue "guessing" even after the game is over. I will fix this in a later version.
